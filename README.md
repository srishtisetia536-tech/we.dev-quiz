# 🧠 Quiz Application

This is a simple and interactive **Quiz Application** built using **HTML, CSS, and JavaScript**.
It allows users to answer multiple-choice questions and view their score at the end.

---

## 🚀 Features

* Multiple-choice questions
* Real-time answer selection
* Score calculation
* Simple and user-friendly interface
* Instant result display

---

## 🛠️ Technologies Used

* **HTML** – Structure of the quiz
* **CSS** – Styling and layout
* **JavaScript** – Logic and interactivity

---

## 📁 Project Structure

```
quiz-app/
│── index.html     # Main quiz interface
│── style.css      # Styling file
│── script.js      # Quiz logic
```

---

## ⚙️ How It Works

1. User starts the quiz
2. Questions are displayed one by one
3. User selects an answer
4. Score is calculated based on correct answers
5. Final result is shown at the end

---

## 💡 How to Run

1. Download or clone the repository
2. Open the folder
3. Open `index.html` in your browser
4. Start the quiz
